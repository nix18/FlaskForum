# FlaskForum
一个简易的基于Flask框架的论坛系统

# 安装步骤

     1、配置Python3.6以上环境（Ubuntu系（自带3.5.3不适配）：需编译安装3.7.3 https://linux265.com/news/3664.html RHEL系：https://www.openos.org/post/162/ ）

     2、pip3 install -r requirements.txt

     3、数据库导入mybbs.sql

     4、修改config.py中的数据库连接信息

     5、运行app.py
